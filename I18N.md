# I18N Policy

[中文](./I18N.zh-CN.md) | **English**

## 1) Scope

This repository is maintained in two languages:

1. English (`en`) as the default file (`*.md`)
2. Simplified Chinese (`zh-CN`) as mirror (`*.zh-CN.md`)

## 2) File Naming Convention

1. Primary file: `*.md` (English)
2. Chinese mirror: `*.zh-CN.md`
3. Root entry: `README.md` (English-first with language switch)
4. For each major document, keep a paired bilingual file when possible.

## 3) Synchronization Rules

1. Update English source first.
2. Update the Chinese mirror in the same PR or a follow-up PR.
3. In translation PR description, include source file and source commit/PR reference.
4. Keep section numbering, formulas, and field names semantically aligned.
5. For high-risk sections (model data, formulas, policy constraints), semantic fidelity is higher priority than style.

These projects cover common patterns:

2. mirrored file structure
3. translation sync workflow (manual or Crowdin-based)
4. scoped docs organization

## 5) Practical Rule In This Repository

Current approach is intentionally lightweight:

1. bilingual file pairs (`.md` + `.zh-CN.md`)
2. no forced migration to `docs/en` + `docs/zh-CN` yet
3. revisit structure when scale and contributor volume increase

## 6) Review Checklist

1. Do links point to same-language targets when available?
2. Are dates, numbers, model names, and formulas consistent?
3. Is anonymization preserved in both languages?
4. Did any untranslated paragraph change policy meaning?

## 7) Language Integrity Gate (Required)

1. Every `*.md` file must be English-first content, not mixed-language or Chinese-first content.
2. Every major `*.md` file must have a matching `*.zh-CN.md` mirror (or explicitly document why it is temporarily missing in the PR description).
3. If a PR changes a source file's language status (for example accidental overwrite), the PR must fix it in the same change set.
4. If this gate fails, do not merge. Fix language attribution and mirror consistency first.
5. For each new `*.md`, is English kept as the primary language and is there a `*.zh-CN.md` mirror?
